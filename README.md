# 🔐 Firebase-Auth

A modern **Firebase Authentication web app** that allows users to register, log in, and authenticate using **Email/Password, Google OAuth, and Phone Number** — built with plain HTML, JavaScript, and Firebase. 

🔗 **Live Demo:** [https://singular-mermaid-c87205.netlify.app/](https://singular-mermaid-c87205.netlify.app/)  
   

---

## 🚀 Features

* ✅ Signup with Email & Password
* ✅ Login with Email & Password
* ✅ Sign in with Google
* ✅ Phone Authentication using OTP
* ✅ Forgot Password Recovery
* ✅ Sign Out functionality

All powered by **Firebase Authentication** — a secure, scalable user identity solution.  

---

## 📦 Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript**
* **Firebase Auth (Firebase JavaScript SDK)**
 

---
 
 

## 📝 Usage

Once running:

* Visitors can **Sign Up** or **Login**
* Use **Google Login** or **Phone OTP Login**
* Password reset via the **Forgot Password** flow
* Users can **Sign Out** after authentication

---

## 🛠️ How It Works

This application relies on **Firebase Authentication** — which provides backend services and ready-to-use SDKs to authenticate users securely using email/password or federated identity providers like Google.  

Firebase handles:

* User creation & login
* Session management
* Phone number OTP verification
